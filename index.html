About the Project

Machine Learning for Humans on Cats is a practical guide to understanding the magic world of AI. Whether you're new to ML or just tired of boring tutorials, this project was made to help you learn with a smile.

Created by Dr. Kateryna Kononova, Professor at Karazin University and expert in data science, this project blends academic depth with a cat-fueled sense of humor to make ML accessible to everyone.

It all began when ChatGPT-3 first became available, and I jumped in. The book came first, followed by Shorts just months later, when tools were still fresh and evolving rapidly. The early videos? Not perfect. But the idea? Still purring strong.

Now the project includes:

  • A beginner-friendly book that explains ML with the help of clever cats
  • YouTube Shorts on every topic
  • One-pagers with code, tips, and easy prompts
  • The code is on GitHub, so you can easily access and experiment with it
